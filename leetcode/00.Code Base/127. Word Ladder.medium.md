##127. Word Ladder - medium - https://leetcode.com/problems/word-ladder/
```
Given two words (beginWord and endWord), and a dictionary's word list, find the length of shortest transformation sequence from beginWord to endWord, such that:

    Only one letter can be changed at a time.
    Each transformed word must exist in the word list.

Note:

    Return 0 if there is no such transformation sequence.
    All words have the same length.
    All words contain only lowercase alphabetic characters.
    You may assume no duplicates in the word list.
    You may assume beginWord and endWord are non-empty and are not the same.

Example 1:

Input:
beginWord = "hit",
endWord = "cog",
wordList = ["hot","dot","dog","lot","log","cog"]

Output: 5

Explanation: As one shortest transformation is "hit" -> "hot" -> "dot" -> "dog" -> "cog",
return its length 5.

Example 2:

Input:
beginWord = "hit"
endWord = "cog"
wordList = ["hot","dot","dog","lot","log"]

Output: 0

Explanation: The endWord "cog" is not in wordList, therefore no possible transformation.

```
### Solution - One Way BFS - TC: O(M×N), SC: O(M×N) - M is the length of the word and N is the number of word in the wordList
```
    public int ladderLength(String beginWord, String endWord, List<String> wordList) {
        if (!wordList.contains(endWord)) return 0;
        Set<String> wordBank = new HashSet<>(wordList);
        Queue<String> queue = new ArrayDeque<>();
        queue.add(beginWord);
        wordBank.remove(beginWord);
        int sequence = 1;
        while (!queue.isEmpty()) {
            int size = queue.size();
            sequence++;
            for (int i = 0; i < size; i++) {
                char[] chars = queue.poll().toCharArray();
                for (int j = 0; j < chars.length; j++) {
                    char origin = chars[j];
                    for (char ch = 'a'; ch <= 'z'; ch++) {
                        chars[j] = ch;
                        String word = new String(chars);
                        if(endWord.equals(word)) return sequence;
                        if(wordBank.contains(word)){
                            wordBank.remove(word);
                            queue.add(word);
                        }
                    }
                    chars[j] = origin;
                }

            }
        }
        return 0;
    }
```

### Solution(Faster) - Two-ended BFS - Worst: TC: O(M×N), SC: O(M×N) 
```
    /*
        two-way BFS
     */
    public int ladderLength(String beginWord, String endWord, List<String> wordList) {
        Set<String> wordSet = new HashSet<>(wordList);
        if (!wordSet.contains(endWord))
            return 0;
        Set<String> left = new HashSet<>(), right = new HashSet<>(), visited = new HashSet<>();
        left.add(beginWord);
        right.add(endWord);
        visited.add(beginWord);
        int step = 0;
        while (!left.isEmpty() && !right.isEmpty()) {
            step++;
            // choose smaller set
            if (left.size() > right.size()) {
                Set<String> tmp = left;
                left = right;
                right = tmp;
            }

            Set<String> tmpLeft = new HashSet<>();
            for (String word : left) {
                char[] chs = word.toCharArray();
                for (int i = 0; i < chs.length; i++) {
                    char origin = chs[i];
                    for (char c = 'a'; c <= 'z'; c++) {
                        if (origin == c)
                            continue;
                        chs[i] = c;
                        String val = String.valueOf(chs);
                        if (!wordSet.contains(val))
                            continue;
                        if (right.contains(val))
                            return step + 1;
                        if (visited.contains(val))
                            continue;
                        tmpLeft.add(val);
                        visited.add(val);
                    }
                    chs[i] = origin;
                }
            }
            left = tmpLeft;
        }
        return 0;
    }
```
